# Data hazırlığı
- Əvvəl dataset-i yüklədim və yoxladım.
- Order Date və Ship Date sütunlarını datetime formatına çevirdim
- Saat analizi üçün Hour sütunu yaratdım
- İl, ay və gün kimi əlavə sütunlar çıxardım
# Ümumi satış və sifariş analizi
- Mən məlumatları müxtəlif qruplara bölüb analiz etdim:
- Region üzrə satışlara baxdım və fərqli regionlarda satışların dəyişdiyini gördüm
- Category üzrə satışları analiz etdim və bəzi məhsulların daha çox satış gətirdiyini müəyyən etdim
- Segment üzrə müştəri davranışını müqayisə etdim və fərqlər olduğunu gördüm
# Vizuallaşdırma
Qrafikləri hazırladım:
- Category üzrə satışların bar chart
- Segment üzrə satışların pie chart
- Saat üzrə satışların line chart
